# ES Lint
## 정의 📋
- 자바스크립트 코드에서 문제를 찾고 고치는 도구

## 장점 👍
- 잠재적인 에러 가능성들을 코드레벨에서 줄여줌
- 코드의 자동 완성
- 자동으로 코드 포맷팅 및 정리

## 사용법 🛠
1. eslint 설치
  ```
  npm i -D eslint
  ```

2. eslint 실행
  ```
  npm run eslint file.js
  ```

### plugins
- ES Lint 자체 여러가지 옵션 외 추가 옵션
